## Visuals Guide (How To Add Cool Stuff)

### 1) Add a top banner image

Create an `assets/` folder in your profile repo and add `banner.png`.

```md
<p align="center">
  <img src="./assets/banner.png" alt="Siddharth Roy banner" />
</p>
```

Tip: Banner size around `1280x320` works well.

### 2) Add project screenshots or GIFs

In each project README:

```md
## Preview
![Dashboard](./assets/dashboard.png)
```

For GIF demos:

```md
![Demo](./assets/demo.gif)
```

### 3) Add badges

You can generate badges from `https://shields.io`.

Example:

```md
![License](https://img.shields.io/badge/license-MIT-green)
![Made with Python](https://img.shields.io/badge/Made%20with-Python-blue)
```

### 4) Add contribution snake (optional visual)

Add this in README:

```md
![snake gif](https://github.com/Siddharth11Roy/Siddharth11Roy/blob/output/github-contribution-grid-snake.svg)
```

Then create a GitHub Action in profile repo to generate it automatically.

### 5) Keep it clean

- Use short sections.
- Keep to 1-2 lines per bullet.
- Put your best projects first.
- Avoid adding too many animated widgets.
